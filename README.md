# Clinko

A simple AFK game for the web built with the create-t3-app. Start with zero buildings or upgrades and work your way to making millions of points per second.

## Tools and Libraries

- create-t3-app
  - Next.js
  - TypeScript
  - Tailwind CSS
  - Prisma
  - tRPC
  - NextAuth.js
- Matter.js
- Zod
